# Color Palette Generator

A simple and evolving Color Palette Generator built to sharpen my front‑end and back‑end development skills. The project allows users to generate color palettes, copy color values, and explore combinations for design inspiration. I’m actively improving both the UI and the underlying logic as I continue learning and refining my workflow.


## 🚧 Current Status

This project is a work in progress. I’ve completed the core structure and functionality, and I’m now making improvements to both the front‑end and back‑end.
